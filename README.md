# VC 足球经理

简化版足球经理游戏（网页版），灵感来自 Football Manager。

## 在线游玩（通勤 / 手机）

**https://as7er.github.io/vc-football-manager/**

- 手机浏览器打开即可，电脑不用开
- 进度存在**当前浏览器**本地（localStorage，**3 个存档槽**）
- 换设备请用游戏内 **导出 / 导入存档**（清缓存会丢进度）
- 建议「添加到主屏幕」，更像 App
- 手机端：页签可横滑；可用 **推进到赛季末**（遇我方比赛会停）

仓库：https://github.com/as7er/vc-football-manager

## 本地运行

ES Module 需要通过 http 打开（不要直接双击 `index.html`）。

### Python
```bash
python -m http.server 8080
```
浏览器打开：http://localhost:8080

### Node
```bash
npx serve .
```

## 已实现内容（持续更新）

- 三级联赛（超联 / 甲级 / 乙级）升降级
- 球员属性、体能、士气、伤病、潜力
- 阵型战术、快速 / 直播比赛
- 转会、自由球员、AI 转会
- 青训、职员、合同、联赛杯
- 董事会目标、球探模糊估值
- 球衣样式与号码、训练日程
- 自动存档、导出 / 导入、推进到比赛日

## 操作提示

1. 选队 → 开始新赛季  
2. **推进一天** 或 **推进到比赛日**  
3. **进入比赛** → 快速或直播模拟  
4. 「训练」调重点/强度，「战术」调阵型  
5. 随时 **存档**；换设备用 **导出/导入**
